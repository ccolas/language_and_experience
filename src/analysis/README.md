# Analysis Scripts

This directory contains scripts for analyzing experimental results and generating visualizations.

## Scripts

### multi_cond_plots.py

Generates the main learning curve comparison figures for the paper, showing performance across different experimental conditions.

**Usage:**
```bash
python multi_cond_plots.py --data-dir /path/to/data --plot-dir /path/to/plots
```

**Inputs:**
- Extracted experiment results (pickle files)
- Human experiment data (JSON files)

**Outputs:**
- Learning curve plots (PNG and PDF) for each game
- Aggregated comparison plots across all games

## Data Requirements

These scripts require experiment results that are generated by running the main experiments. The expected directory structure is:

```
data/inference_data/
├── machine_no_feedback/          # Individual learning results
│   ├── beesAndBirds/
│   │   ├── 0/                    # Seed 0
│   │   │   ├── params.json
│   │   │   └── dumps/
│   │   ├── 1/                    # Seed 1
│   │   └── ...
│   └── ...
└── machine_human_feedback/       # Social learning with human messages
    └── ...
```
